== Description ==
Igentworks Auto Blogs is a powerful WordPress plugin that automates content creation using advanced AI. It allows you to generate high-quality blog posts on a predefined schedule, keeping your website fresh and engaging without manual effort.

This plugin is perfect for websites that need regular updates and fresh content, but donâ€™t have the time or resources to write blog posts manually. Igentworks Auto Blogs simplifies content creation, making it easy for you to focus on growing your website and audience.

== Features ==
- Automatically generates blog posts using advanced AI
- Customizable scheduling options for post generation
- Supports multiple content categories
- Easy-to-use interface with no coding required
- Seamless integration with WordPress

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/igentworks-auto-blogs/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings by navigating to **Settings > Igentworks Auto Blogs** in the WordPress admin dashboard.
4. Set up the schedule and category preferences for content generation.

== Frequently Asked Questions ==

= How do I set up content generation? =
Once the plugin is activated, go to the plugin settings page in the WordPress dashboard. You can set up the schedule and choose the categories for which the content will be generated.

= Can I customize the generated content? =
Yes, the plugin allows you to define the parameters for the content, such as the tone, subject, and keywords. You can also fine-tune the output before publishing.

= Is this plugin compatible with all themes? =
Yes, Igentworks Auto Blogs is compatible with most WordPress themes. However, some custom themes may require additional configuration.
